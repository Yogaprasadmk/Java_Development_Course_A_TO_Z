```js
int a = Sc.nextInt();
```
<p>Float:</p>

```js
Float a= Sc.nextFloat();
```
<p>Byte:</p>

```js
Byte a = Sc.nextByte();
```
<p>Short:</p>

```js
Short a = Sc.nextShort();
```
<p>String:</p>

One word:
```js
String s = Sc.next();
```
or
Two or more words or entire line 

```js
String S = Sc.nextLine();
```

```js
Char t = Sc.next.charAt();
```

